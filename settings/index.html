云端同步（推荐）

使用 GitHub 私密 Gist 存你的个人数据（不进公开仓库）。 配置一次后，各模块改动约 1.5 秒自动上传；新设备打开任意页面会先拉取最新。

如何获取 Token？
GitHub → Settings → Developer settings → Personal access tokens → Generate（classic）→ 勾选 gist 权限即可。 Token 只保存在本机浏览器，不会提交到 git。

已有 Gist 可填 ID 复用；换电脑时填同一 ID + 同一 Token。

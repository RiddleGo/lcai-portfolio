# 评分与评级说明

研判 **模式** 见 [`02-成长与价值双模式.md`](02-成长与价值双模式.md)。下表默认 **价值型**；成长型 L1–L3 阈值见该文档第三节。

## 分层权重

| 层级 | 权重 | 门禁（价值型） |
|------|------|----------------|
| L0 | — | 硬指标全过才继续 |
| L1 生意 | 30% | 软分 ≥70 |
| L2 财务 | 25% | 软分 ≥75 |
| L3 估值 | 25% | 软分 ≥65，安全边际硬指标 ≥25% |
| L4 执行 | 10% | — |
| L5 行业 | 10% | — |

成长型：L3-01 硬指标 ≥15%（Forward）；L1-05、L3-04 权重加重。

## 评级

| 评级 | 总分 | 含义 |
|------|------|------|
| A | ≥80 | 可建仓 |
| B | 72–79 | 观察池，接近建仓 |
| C | 60–71 | 仅跟踪 |
| D | <60 | 不符合标准 |

## 判定

| 判定 | 条件（价值型） |
|------|----------------|
| 买入 | 无否决 + L1–L3 硬指标全过 + 总分≥72 + 安全边际≥25% |
| 观察 | 硬指标未全过或估值不够 |
| 持有 | 已持仓 + 逻辑未破 |
| 减仓 | 已持仓 + L3 弱或 PE 极端 |
| 卖出 | 已持仓 + 硬指标证伪 |
| 排除 | 否决项或 L0 Fail |

成长型：买入需 Forward 安全边际 ≥15% + 成长信号 ≥2 + thesis；卖出看 thesis 触发器。

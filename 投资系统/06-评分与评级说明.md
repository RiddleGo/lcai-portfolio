# 评分与评级说明

## 分层权重

| 层级 | 权重 | 门禁 |
|------|------|------|
| L0 | — | 硬指标全过才继续 |
| L1 生意 | 30% | 软分 ≥70 |
| L2 财务 | 25% | 软分 ≥75 |
| L3 估值 | 25% | 软分 ≥65，安全边际硬指标 ≥25% |
| L4 执行 | 10% | — |
| L5 行业 | 10% | — |

## 评级

| 评级 | 总分 | 含义 |
|------|------|------|
| A | ≥80 | 可建仓 |
| B | 72–79 | 观察池，接近建仓 |
| C | 60–71 | 仅跟踪 |
| D | <60 | 不符合标准 |

## 判定

| 判定 | 条件 |
|------|------|
| 买入 | 无否决 + L1–L3 硬指标全过 + 总分≥72 + 安全边际≥25% |
| 观察 | 硬指标未全过或估值不够 |
| 持有 | 已持仓 + 逻辑未破 |
| 减仓 | 已持仓 + L3 弱或 PE 极端 |
| 卖出 | 已持仓 + 硬指标证伪 |
| 排除 | 否决项或 L0 Fail |

## 仓位建议

| 评级 | 单票上限 |
|------|----------|
| A | 25% |
| B | 10% |
| C/D | 0%（不新建仓） |

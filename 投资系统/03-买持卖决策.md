# 买 / 持 / 卖 决策

与 `screen-engine.js` 判定逻辑一致。

## 观察池

- L0 否决或 L0 硬指标 Fail → **排除**
- L1–L3 任一硬指标 Fail → **观察**
- 总分 <72 或安全边际 <25% → **观察**

## 建仓（买入）

- 无否决
- L1–L3 硬指标全 Pass
- 总分 ≥72，评级 A/B
- L3-01 安全边际 ≥25%
- 心理/能力圈勾选通过

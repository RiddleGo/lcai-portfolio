# 买 / 持 / 卖 决策

与 `screen-engine.js` 判定逻辑一致；**成长型**阈值见 [`02-成长与价值双模式.md`](02-成长与价值双模式.md)。

## 模式选择（建仓前）

1. 默认 **价值型**
2. 成长型须：L0 无否决 + 能力圈 + 书面 thesis + 成长信号 ≥2
3. 两种模式都不过 → 观察或排除

## 观察池

- L0 否决或 L0 硬指标 Fail → **排除**
- L1–L3 任一硬指标 Fail（按所选模式）→ **观察**
- 总分 <72 或安全边际未达模式要求 → **观察**

## 建仓（买入）

### 价值型

- 无否决
- L1–L3 硬指标全 Pass
- 总分 ≥72，评级 A/B
- L3-01 安全边际 ≥25%
- 心理/能力圈勾选通过

### 成长型

- 无否决（含 L0-06、L3-05，不可豁免）
- 成长信号 ≥2 + thesis 已填写
- L1–L3 硬指标全 Pass（成长阈值）
- 总分 ≥72，评级 A/B
- L3-01 Forward 安全边际 ≥15%
- 心理/能力圈勾选通过

## 已持仓

| 判定 | 价值型 | 成长型 |
|------|--------|--------|
| 持有 | 逻辑未破，L3 尚可 | thesis 核心假设仍成立 |
| 减仓 | L3 得分 <50 或 PE 极端 | Forward MOS <0 或 PEG >2 |
| 卖出 | L1/L2 硬指标 Fail 或否决 | 卖出触发器命中 / 硬指标 Fail / 否决 |
| 排除 | L0 否决 | 同左 |

## Thesis 模板（建议买入前填写）

见 [`02-成长与价值双模式.md`](02-成长与价值双模式.md) 第六节。价值型至少含：

1. 核心假设（3 条）
2. 护城河类型
3. 安全边际测算
4. 卖出触发（3 条）

成长型另须：成长信号勾选、Forward 估值、3 条可量化卖出触发。

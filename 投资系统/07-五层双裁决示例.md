## 北方华创 002371

### 五层定位

| 字段 | 内容 |
|------|------|
| **AI 层** | **L2 芯片** |
| **环节** | 半导体设备（刻蚀/沉积/CVD 平台型） |
| **瓶颈强度** | **binding**（管制下国产化加速） |
| **研究分层** | **§8.6 核心三角 #1** |
| **国产替代** | **突破中** — 平台型设备追赶，最高长周期壁垒 |
| **产业阶段** | L2 **爆发期** |

### LCAI 价值型（引擎默认）

| 项目 | 结果 |
|------|------|
| 判定 | **排除** |
| 评级 / 总分 | B / 76 |
| 否决 | **L3-05 极端高估值**（PE 94，利润 YoY 3.4%） |
| 硬 Fail | L2-01 OCF/EPS 0.45 · L3-01 安全边际 · L3-02 PE |
| 现价 / 公允 | 721 元 / 229 元（EPS×30） |

**优势**: ROE 五年 16% · 5/5 盈利 · 营收 +25.8% · 毛利率 40%  
**短板**: 现金流弱（设备厂常见，但规则不豁免）· 估值与增速严重不匹配

### LCAI 成长型（二次裁决）

| 成长信号 | 结果 |
|----------|------|
| G1 营收高增 | ✓ +25.8% |
| G2 盈利拐点 | ✗ 利润 YoY 仅 +3.4% |
| G3 投入期优质 | ✓ 毛利率 40% |
| G4 现金流改善 | ✗ OCF/EPS 0.45 |

| 成长硬指标 | 结果 |
|------------|------|
| L0 否决 | **L3-05 仍触发**（不可豁免） |
| L2-01（≥0.6） | ✗ 0.45 |
| L3-01（Forward ≥15%） | ✗ 回溯模型 -214% |

### 综合结论

| 维度 | 结论 |
|------|------|
| **产业** | L2 国产替代 **最核心标的之一**，逻辑成立 |
| **财务** | 价值/成长 **均排除**（极端估值否决 + 现金流） |
| **表达工具** | **不买个股**；等 PE 回落 + OCF 改善，或用 **588200/159516 芯片 ETF** 表达 L2 |
| **触发再评估** | 利润 YoY 重回 ≥15% · OCF/EPS ≥0.6 · PE ≤50 且无 L3-05 |

## 中际旭创 300308

### 五层定位

| 字段 | 内容 |
|------|------|
| **AI 层** | **L3 基础设施** |
| **环节** | 800G 光模块（AI 工厂「神经网络」） |
| **瓶颈强度** | **binding** |
| **研究分层** | **§8.6 核心三角 #1** |
| **国产替代** | **已规模化** — 全球光模块制造中心 |
| **产业阶段** | L3 **建设高峰** |

### LCAI 价值型

| 项目 | 结果 |
|------|------|
| 判定 | **观察** |
| 评级 / 总分 | **A / 85** |
| 否决 | 无 |
| 硬 Fail | L3-01 安全边际 · L3-02 PE（139 > 40） |
| 现价 / 公允 | 1368 元 / 196 元（default PE×20） |

**优势**: ROE 22% · 5/5 盈利 · 营收 +192% · 利润 +262% · OCF/EPS 0.93 · **PEG 0.53**  
**短板**: 回溯估值模型显示极贵（高成长股常见误分类为 default 行业）

### LCAI 成长型（二次裁决）

| 成长信号 | 结果 |
|----------|------|
| G1 | ✓✓ 营收 +192% |
| G2 | ✓ 5/5 盈利，利润高增 |
| G4 | ✓ OCF 0.93 |
| G5 国产化份额 | ✓ 全球制造中心地位 |

| 成长硬指标 | 结果 |
|------------|------|
| L0 否决 | 无（PE 高但增速 >10%，**未触发 L3-05**） |
| L1-01 / L1-02 | ✓ |
| L2-01 | ✓ |
| L3-01（≥15%） | ✗ 回溯 -598%；**Forward 粗算**：EPS 若按 +80% → ~17.6，公允 PE 28 → ~493 元，仍远低于现价 → **未过 15% 边际** |
| L3-02 | ✗ PE 仍超半导体上限 50 |

### 综合结论

| 维度 | 结论 |
|------|------|
| **产业** | L3 **最最最核心**，国产替代 **已规模化**，产业逻辑 **强** |
| **财务** | 质量 **A 级**，但 **任何模式均未达买入线** |
| **表达工具** | **观察池**；回调至 Forward 边际 ≥15% 再建；或 **159819 AI ETF** 部分表达 L3–L4 β |
| **跟踪** | 800G 出货量 · 毛利率 · 下季 EPS 指引 |

## 赛力斯 601127（已持仓）

### 五层定位

| 字段 | 内容 |
|------|------|
| **AI 层** | **L5 应用** |
| **环节** | 新能源车 / 智驾（华为链叙事） |
| **瓶颈强度** | structural |
| **研究分层** | **非 §8.6 主线**（智驾核心是 **德赛西威 002920**） |
| **国产替代** | 突破中（域控/整车国产化） |
| **产业阶段** | L5 商业化初期 |

### LCAI 价值型

| 项目 | 结果 |
|------|------|
| 判定 | **观察**（已持仓） |
| 评级 / 总分 | B / 75 |
| 硬 Fail | L1-01 ROE -4.8% · L1-02 仅 2/5 盈利 · L3-01 边际 11.9% |
| 持仓 | 3000 股 · 成本 164.7 · 现价 64.8 · **浮亏 ~61%** |

**优势**: 营收 +34% · OCF/EPS 4.15（现金流强）· PE 17.6 不贵  
**短板**: 五年 ROE 负 · 盈利历史不稳定 · **不在五层智驾主线**

### LCAI 成长型（二次裁决）

| 成长信号 | 结果 |
|----------|------|
| G1 | ✓ 营收 +34% |
| G2 | △ 2/5 盈利年；需确认 **最近 2 年连续盈利**（若满足则 ✓） |
| G4 | ✓ OCF 强 |

| 成长硬指标 | 结果 |
|------------|------|
| L1-01（≥10% 或拐点） | ✗ 五年 ROE 均值仍 -4.8% |
| L1-02（≥2 年且近 2 年连续） | △ 取决于最近两期是否均盈利 |
| L3-01（≥15%） | ✗ 11.9%，差 3ppt |

### 综合结论

| 维度 | 结论 |
|------|------|
| **产业** | L5 有故事，但 **非研究框架首选**；与德赛西威相比纯度/PMF 验证弱 |
| **财务** | 价值/成长 **均未过建仓线**；深套不宜「等解套」当 thesis |
| **持仓动作** | **不加仓**；写华为链 thesis + 下季财报；若 L1 仍 Fail 或问界增速放缓 → **列入减仓/退出**（与还债可合并） |
| **表达工具** | 若坚持 L5 智驾，研究 **002920** 是否过成长线，而非加仓赛力斯 |

## 三只标的横向对比

|  | 北方华创 | 中际旭创 | 赛力斯 |
|--|---------|---------|--------|
| 五层 | L2 芯片 | L3 基础设施 | L5 应用 |
| §8.6 | 核心 | 核心 | **非主线** |
| 国产替代 | 突破中 | 已规模化 | 突破中 |
| 价值 LCAI | **排除** | 观察 | 观察 |
| 成长 LCAI | **排除** | 观察 | 观察 |
| 能否买 | 否 | 等价格/ETF | 否（持仓仅观察/规划退出） |
| 优先表达 | 芯片 ETF | 观察池/AI ETF | 减仓释额度 |

**总括**：五层研究告诉你 **「该看哪里」**；LCAI 告诉你 **「现在能不能买」**。当前三只 **无一达买入线**——这与「国产替代/AI 很重要」不矛盾，说明 **好赛道也要好价格 + 好财报**。

## 对你持仓的启示

1. **英维克**（L1 核心）与 **北方华创** 同类问题：产业对、**LCAI 否决/排除**。  
2. **缺 L2–L3 核心暴露**：中际/北方华创都在观察，应用 **ETF 计划** 比乱买个股更一致。  
3. **赛力斯**：五层 L5 非主线 + 深套 → 还债阶段 **优先减**，勿因叙事加仓。

# 成长与价值双模式

> 取代「个股轨 + ETF 轨」的偶然分工。全系统只有 **一套宪法**、**两种研判模式**；ETF 是表达工具，不是第三条规则。

## 总流程

```
新标的 / 加仓
    ↓
L0 门禁（两种模式共用，否决项不可豁免）
    ↓
选择模式：默认价值型；成长型需书面 thesis
    ↓
L1–L3 按所选模式阈值研判
    ↓
判定 → 表达工具（个股 or ETF）
```

---

## 一、模式定义

### 价值型（默认）

**适用**：商业模式成熟、盈利稳定、现金流可验证，等好价格。

| 维度 | 标准 |
|------|------|
| 时间视角 | 回溯 5 年财报 |
| 核心问题 | 这门生意是否持续高 ROE？现在够便宜吗？ |
| 估值主轴 | 历史 EPS × 行业公允 PE；简化 DCF 交叉验证 |
| 典型标的 | 消费龙头、金融、成熟医药、稳定 SaaS（已盈利多年） |

### 成长型（需主动选择）

**适用**：行业空间大（国产替代、AI 算力、半导体设备等），公司仍处投入或扭亏期，但 **增长可被数据验证**。

| 维度 | 标准 |
|------|------|
| 时间视角 | 近 2–3 年趋势 + 未来 1–2 年前瞻 |
| 核心问题 | 增长 thesis 是否成立？价格是否为增长留了合理空间？ |
| 估值主轴 | Forward EPS / PEG / 简化 DCF（增长假设有上限） |
| 典型标的 | 扭亏车链、半导体设备、AI 基础设施（**已过 L0、未触发否决**） |

**不是成长型**：纯概念、无营收验证、触发 L0-06/L3-05 否决的叙事票（如极高 PE + 利润崩塌）。

---

## 二、模式选择（建仓前必做）

默认 **价值型**。仅当以下 **全部满足** 才可切换成长型：

1. **L0 无否决**（ST、造假嫌疑、异常票、极端泡沫否决均不可豁免）
2. **能力圈确认**（L0-01 勾选，能写清 3 年竞争格局）
3. **成长 thesis 书面化**（用下方模板）
4. **成长信号 ≥2 条**（见下表）

### 成长信号（至少 2 条）

| # | 信号 | 阈值 |
|---|------|------|
| G1 | 营收增速 | 近 2 期 YoY 均 ≥15%，或行业 TAM 扩张有公开数据支撑 |
| G2 | 盈利拐点 | 最近 2 年连续盈利，或利润 YoY 连续 2 期改善 |
| G3 | 投入期优质 | ROE 暂 <15%，但毛利率 ≥25% 且护城河代理 ≥60 |
| G4 | 现金流改善 | OCF/EPS 较 3 年前提升，或 OCF 为正且 YoY 改善 |

**切换失败**：成长信号不足 → 只能走价值型；价值型也不过 → **观察或排除**，不得用「AI/国产替代」强行归类为成长。

## 三、L0–L3 阈值对照

### L0 门禁 — 两种模式完全相同

| 编号 | 价值型 | 成长型 | 说明 |
|------|--------|--------|------|
| L0-01 能力圈 | 人工确认 | 人工确认 | 成长型要求更高：须能解释行业 3 年格局 |
| L0-02 非 ST | 硬 | 硬 | |
| L0-03 流动性 | 硬 | 硬 | |
| L0-05 造假嫌疑 | **否决** | **否决** | 不可豁免 |
| L0-06 异常票 | **否决** | **否决** | 不可豁免 |

### L1 生意 — 成长型部分放宽

| 编号 | 价值型 | 成长型 |
|------|--------|--------|
| L1-01 ROE 均值 | 5 年均 ≥15% | **5 年均 ≥10%** 或 **最近年度 ≥15%** 或 **3 年 ROE 趋势向上** |
| L1-02 盈利稳定 | 5 年中 ≥4 年盈利 | **5 年中 ≥2 年盈利** 且 **最近 2 年连续盈利** |
| L1-05 利润增长 | 软 ≥0% | **软 ≥10%**（加重权重） |
| L1-06 利润崩塌 | **否决** | **否决** |

### L2 财务 — 成长型略放宽，否决不变

| 编号 | 价值型 | 成长型 |
|------|--------|--------|
| L2-01 现金流质量 | OCF/EPS ≥0.8 | **≥0.6** 或 **OCF 为正且 YoY 改善** |
| L2-03 盈利趋势 | 软 ≥0% | 软 **≥5%** |
| L2-04 现金流背离 | **否决** | **否决** |

### L3 估值 — 成长型用前瞻，底线仍存

| 编号 | 价值型 | 成长型 |
|------|--------|--------|
| L3-01 安全边际 | 历史 EPS，**≥25%** | Forward EPS（下财年或 2 年 CAGR 折现），**≥15%** |
| L3-02 PE 合理 | ≤行业上限 | ≤行业上限（半导体 50、软件 45 等不变） |
| L3-04 PEG | 软 ≤2 | 软 **≤1.5**（成长型加重） |
| L3-05 极端估值 | **否决** | **否决**（PE>80 且增速<10% 一律不买） |
| L3-06 DCF | 交叉验证 | 交叉验证；增长假设 g 上限 25%，WACC 下限 10% |

### L4 / L5 — 成长型仓位更保守

| 项目 | 价值型 | 成长型 |
|------|--------|--------|
| 单票上限 A | 25% | **15%** |
| 单票上限 B | 10% | **8%** |
| 成长型总仓上限 | — | 证券资产 **≤40%**（含成长型个股 + 科技 ETF） |
| 核心仓数量 | ≤5 | ≤5（价值 + 成长合计） |

## 四、判定与买持卖

### 建仓（买入）

**价值型**：无否决 + L1–L3 硬指标全过 + 总分 ≥72 + L3-01 ≥25% + 心理/能力圈通过

**成长型**：无否决 + 成长信号 ≥2 + L1–L3（成长阈值）硬指标全过 + 总分 ≥72 + L3-01 ≥15% + thesis 已填写

### 已持仓

| 判定 | 价值型 | 成长型 |
|------|--------|--------|
| 持有 | 逻辑未破，L3 尚可 | **thesis 3 条核心假设仍成立** |
| 减仓 | L3 得分 <50 或 PE 极端 | Forward MOS <0，或 PEG >2 |
| 卖出 | L1/L2 硬指标 Fail 或否决 | **任一卖出触发器命中**（见 thesis 模板） |
| 排除 | L0 否决 | 同左；**成长标签不能豁免** |

### 两种模式都 Fail

→ **观察**（硬指标部分未过）或 **排除**（否决 / L0 Fail）

## 五、表达工具：个股 vs ETF

模式决定 **用什么标准**；工具决定 **怎么表达**。

| 情形 | 建议工具 |
|------|----------|
| 价值型 + 高确信 + 过线 | **个股** |
| 成长型 + 看懂赛道、龙头未过线或分散 | **ETF**（见 [`09-ETF分批计划.md`](09-ETF分批计划.md)） |
| 成长型 + 单票过成长线 + 高确信 | **个股** |
| 两种模式都未过 | **不买**；不因 ETF 绕过个股标准（ETF 仍受成长型总仓 40% 约束） |

科技 ETF 计划（科创50、芯片、AI、机器人）归属 **成长型表达工具**，计入成长型总仓 40% 上限。

## 八、与宪法的关系

- 宪法第 2 条「安全边际」：价值型 25%；成长型 15%（Forward），**均有硬底线**
- 宪法第 4 条「集中有限」：成长型单票更低上限，总成长仓 ≤40%
- 宪法第 7 条「财报为准」：成长型更看 OCF 改善，**叙事不能替代 G1–G4 信号**
- 宪法第 9 条「禁止 FOMO」：成长型必须 thesis + 卖出触发，否则视为 FOMO
- **产业主题**：国产替代 / AI 五层 **不能豁免** L0 否决与 L3-05 极端泡沫

详见 [`00-投资宪法.md`](00-投资宪法.md) 学派冲突裁决表。

## 九、AI 五层产业框架（成长型 L5 overlay）

> 深度研究：[`docs/research/`](../docs/research/README.md) — [04 总览](../docs/research/04_ai_five_layers_analysis.md) · [10 全链映射](../docs/research/10_full_chain_industry_mapping.md) · [05–09 深度](../docs/research/05-09_layers_index.md)

### 9.1 五层定义与传导

```
第1层 能源     → 电力/散热/算电协同（硬约束）
第2层 芯片     → GPU/HBM/CoWoS/设备/国产算力
第3层 基础设施 → AI 工厂：服务器/光模块/IDC/AIDC
第4层 模型     → LLM/语料/MaaS/订阅型 AI 软件
第5层 应用     → Agent/智驾/安防/金融 IT/机器人 PMF
        ↑ 应用成功向下拉动全栈
```

**中国路径**：算电协同 + 东数西算 + 枢纽绿电；**不复制** 美国 onsite 燃气发电主线。

### 9.2 产业阶段（2025–2026）

| 层 | 阶段 | 投资含义 |
|----|------|----------|
| L1 能源 | **瓶颈期** | binding 优先（液冷、算电协同） |
| L2 芯片 | **爆发期** | 设备/封装/国产算力；警惕周期 |
| L3 基础设施 | **建设高峰** | 光模块/服务器/AIDC；盯 Capex ROI |
| L4 模型 | **门槛跨越** | A 股映射平台+语料+订阅，非 frontier 本体 |
| L5 应用 | **商业化初期** | 要 PMF 可验证，非纯主题 |

### 9.3 瓶颈强度

| 等级 | 含义 | 示例 |
|------|------|------|
| **binding** | 无此环节则扩产停摆 | CoWoS、HBM、800G 光模块、液冷、算电协同 |
| **structural** | 结构性偏紧 | ABF 载板、AI 服务器 PCB、半导体设备 |
| **cyclical** | 周期景气 | MLCC、二线 IDC、部分储能 |
| **narrative** | 概念强、瓶颈弱 | 中国燃气轮机链、人形机器人整机 |

成长型优先 **binding / structural + 核心/增强**；**narrative** 默认不建仓。

### 9.4 §8.6 核心三角（研究主线，非买入清单）

| 层 | 核心三角 | 增强 |
|----|----------|------|
| L1 能源 | 国电南瑞 · 英维克 · 思源电气 | 甘肃能源 · 长江电力 |
| L2 芯片 | 北方华创 · 长电科技 · 海光信息 | 澜起科技 · 寒武纪 |
| L3 基础设施 | 中际旭创 · 浪潮信息 · 润泽科技 | 工业富联 · 新易盛 |
| L4 模型 | 科大讯飞 · 海天瑞声 · 金山办公 | 昆仑万维 · 拓尔思 |
| L5 应用 | 德赛西威 · 海康威视 · 恒生电子 | 中科创达 · 绿的谐波 |

§8.6 标的 **仍须逐只过 LCAI**。

### 9.5 五层 × ETF 对齐

| ETF | 覆盖层 | 备注 |
|-----|--------|------|
| 588000 科创50 | L2–L4 宽基 | 打底 |
| 588200 / 159516 | **L2 芯片** | 国产替代主战场 |
| 159819 人工智能 | L4–L5 | 主题 β，非 binding |
| 562500 机器人 | L5 应用 | 具身智能，偏 narrative |

binding 层（L1–L3）优先 ETF 或 §8.6 个股；L4–L5 主题 ETF ≤ 成长总仓一半。

## 十、国产替代：四阶段 × 五层

### 10.1 四阶段

| 阶段 | 成长型怎么处理 |
|------|----------------|
| **已规模化** | 看估值 + 份额天花板（光模块、PCB 制造） |
| **突破中** | **主战场**；跟踪订单、毛利率、国产化率 |
| **受制** | 慎炒概念（EUV、HBM 原厂多无 A 股映射） |
| **不适用** | 资源/模式差异（如美国燃机链） |

### 10.2 分层地图（摘要）

| 层 | 国产替代主战场 | 受制/空白 |
|----|----------------|-----------|
| L1 | 南瑞、液冷、思源、特高压 | 重型燃机 |
| L2 | 北方华创、长电、海光、中微 | EUV、HBM 原厂 |
| L3 | 中际旭创、浪潮、润泽、工业富联 | 高端交换芯片 |
| L4 | 讯飞、金山、语料链 | frontier 未上市 |
| L5 | 德赛西威、海康、恒生 | 代码 Agent 多在美股 |

### 10.3 成长信号增强（G5–G7，可选第 3 条信号）

| 代号 | 信号 | 阈值 |
|------|------|------|
| G5 | 国产化率/份额 | 近 2 期份额提升或头部客户导入 |
| G6 | 订单/backlog | 营收与在手订单方向一致 |
| G7 | 政策契合 | 「人工智能+」/信创赛道 + **财报验证** |

G5–G7 **不能替代** G1–G4。

### 10.4 常炒但未入 §8.6 主线

| 环节 | 代表 | 原因 |
|------|------|------|
| AI 服务器 PCB | 沪电、胜宏 | structural；AI 收入占比待验证 |
| HVLP 铜箔 | 德福、嘉元 | 易混电池箔；非全行业缺 |
| MLCC/电容 | 三环、法拉 | cyclical |
| 二线 IDC | 光环、奥飞 | AI 纯度低 |
| 人形机器人 | 埃斯顿等 | narrative，PMF 早于基本面 |

见 [10 总表 §9](../docs/research/10_full_chain_industry_mapping.md#9-专项深述材料与被动元件)。

## 十一、成长型建仓：五层 × LCAI 联合流程

```
Step 1  五层定位：哪层？瓶颈强度？国产替代阶段？
Step 2  研究分层：核心/增强/延伸/叙事？对照 §8.6
Step 3  模式选择：AI/国产替代默认成长型
Step 4  LCAI 裁决（成长阈值）；否决则停止
Step 5  Thesis：五层字段 + 3 假设 + 3 卖出触发
Step 6  表达：个股过线→个股；赛道对个股不过→ETF
Step 7  仓位：单票 ≤15%/8%；成长总仓 ≤40%
```

### 冲突裁决

| 冲突 | 裁决 |
|------|------|
| §8.6 核心 vs LCAI 排除 | **LCAI 为准** |
| 国产替代叙事 vs L1/L2 Fail | **LCAI 为准** |
| binding 层 vs 高估值 | Forward MOS ≥15%，否则等 |
| 跨层标的（如腾讯） | 按主要利润来源定一层 |

### 去杠杆后理想暴露（方向参考，非买入建议）

| 现状问题 | 理想方向 |
|----------|----------|
| 重仓平安/赛力斯 | 清排除项；赛力斯若留须 L5 thesis |
| 英维克 L1 对但 LCAI 否 | 等财务修复或 ETF/南瑞/思源表达 L1 |
| 缺 L2–L3 暴露 | 588200/159516；过线后再看北方华创/长电/中际 |
| 微盟/京东健康 | 非五层主线，退出释额度 |

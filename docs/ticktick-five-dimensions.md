## 一、整体框架

### 1.1 清单结构

```
收集箱                    ← 临时想法，48 小时内归类或删除

复利人生/
├── 📜 决策               ← 目标、OKR、反思、宪法、重大决策
├── 💰 财务               ← 还债执行 + 投资计划 + 机会观察池
├── 🧠 阅读               ← 精读、读书笔记
├── 🛠 职业               ← 转行学习、项目、每周交付
└── 🏃 健康               ← 训练习惯（你说已建好 ✓）
```

个人备忘、On The Way：并入 **收集箱** 或 **📜 决策**（无日期）。

### 1.2 五维分工

| 维度 | 清单 | 管什么 |
|------|------|--------|
| 📜 心 · 决策 | 决策 | 目标 OKR、总结反思、宪法、卖股/换工作/投资前决策 |
| 💰 钱 · 财务 | 财务 | 现在：还债卖股；以后：定投计划；随时：机会观察池 |
| 🧠 脑 · 阅读 | 阅读 | 精读、认知输入、投资/决策类书籍 |
| 🛠 手 · 职业 | 职业 | 全栈/AI 转型、项目 demo、简历叙事 |
| 🏃 体 · 健康 | 健康 | 训练、睡眠、精力（你已配置习惯） |

### 1.3 每日 / 每周

**每天（2 分钟）**  
打开「今天」→ 习惯打卡 → 按优先级勾选 → 带「写备注」的先写完再勾

**每周日（15 分钟）**  
更新 📜 四条 OKR 进度 → 清空收集箱 → 看 💰 下月还款是否已建任务

**任务 3 · 人生宪法**

标题：`人生宪法 · 重大决策前必读`

备注：
```
本系统以「先活下来、再复利」为轴，10 条不可违背：

1. 阶段优先：2027.4 前平台债清零；只还不投、不开杠杆、不用消费贷买股
2. 执行大于预测：财务待办勾选即信用；错过还款比看错行情更致命
3. 双框架决策：赛力斯不卖；卖股还债须先写决策日记
4. 重大决策必记录：换工作、借新债、单笔卖仓>10%市值、改 OKR — 先写再动
5. 能力圈投资：不懂不投
6. 读→写→用：每季精读 ≤3 本；笔记须对应一条行动
7. 项目即简历：可展示产出 > 证书
8. 精力底座：睡眠与运动是执行力前置；streak 断档时不加大赌注
9. 不做清单：每季 OKR 须含明确不做的事
10. 复盘：每月总结；每季 OKR 复盘

冲突时：人生宪法（偿债、杠杆）优先于任何买入信号。
```

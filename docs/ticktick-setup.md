# 复利人生 · 只用滴答清单

**Russshare 网站可不用。** 执行、OKR 进度、反思笔记、财务勾选、决策记录 — 全部在滴答完成。

## 一键导出完整配置

在仓库根目录运行（仅需 Python，不启动网站）：

```bash
python scripts/export_ticktick.py
```

生成 [ticktick-life-os.md](./ticktick-life-os.md) — 含：

- 10 个清单结构（含 **07-反思笔记**、**08-决策日记**）
- 人生宪法全文（贴进 00-方向 任务备注）
- 本季 4 条 OKR + 子任务
- 财务待办与到期日
- 习惯、阅读、职业每周节奏
- 反思/学习/月总结/季复盘/决策 **备注模板**

把 Markdown 存到手机/电脑，按章节在滴答里建清单即可。

## 每日用法

1. 只打开滴答 **「今天」**
2. 习惯打卡
3. 带 `#笔记` 的任务：先在 **备注** 写好正文，再勾选
4. OKR 进度：改 **01-OKR** 任务标题里的 `(85%)` 数字

## 清单速查

| 清单 | 作用 |
|------|------|
| 00-方向 | 宪法、不做清单（不设日期） |
| 01-OKR | 本季 KR，进度写标题 |
| 02-财务 | 还款卖股 |
| 03-训练 | 习惯 |
| 04-阅读 | 精读提醒 |
| 05-工作 | 职业交付 |
| 06-备忘 | 杂事 |
| 07-反思笔记 | **长文档案**（备注里写） |
| 08-决策日记 | 重大决策先写再动 |
| 99-以后再说 | 远期 |

## 刷新

财务或 OKR 在本地 JSON 改了之后，再跑一遍 `export_ticktick.py`，对照更新滴答里的数字和待办。

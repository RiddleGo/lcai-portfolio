芯片层是 **将能源转化为 AI 可用算力** 的核心环节。NVIDIA 定义：处理器须在大规模并行、高带宽内存（HBM）、快速互连（NVLink/InfiniBand）支持下，高效完成矩阵运算——芯片效率决定 AI **扩展速度**与**智能成本曲线**。

Gartner 2026-04：AI 半导体将占全球半导体收入约 **30%**；2026 年半导体总收入 **1.32 万亿美元**（+64%），为 **20 年来最高增速**。

## 第 1 章 本质定义

### 1.1 芯片层包含什么

| 类别 | 说明 |
|------|------|
| **AI 加速器** | GPU、TPU、训练/推理 ASIC |
| **CPU** | 通用计算、集群管理、部分推理 |
| **HBM 高带宽内存** | 与 GPU 封装在同一模块，决定数据吞吐 |
| **网络芯片** | InfiniBand、以太网交换、DPU |
| **先进封装** | CoWoS、Chiplet、2.5D/3D 封装 |
| **EDA / IP** | 芯片设计与知识产权 |

### 1.2 训练芯片 vs 推理芯片

| 维度 | 训练 (Training) | 推理 (Inference) |
|------|-----------------|------------------|
| 任务 | 学习模型参数 | 使用已训练模型生成输出 |
| 精度 | FP16/BF16/FP8 混合 | INT8/FP8/INT4 量化 |
| 集群规模 | 数千~数万 GPU | 单卡~数百卡 |
| 代表产品 | NVIDIA H100/B200、Google TPU | NVIDIA L40S、Google TPU inference、边缘 NPU |
| 市场趋势 | Hyperscaler 主导 | 2026 起推理算力需求快速上升 |

### 1.3 为何独立成层

- 算力效率是 AI 经济性的 **第一杠杆**
- 供应链高度专业化：设计、制造、封装、内存分离
- 地缘与出口管制直接作用于本层（BIS ECCN 3A090/4A090）
- 与能源层耦合加深：更高算力 = 更高功耗

## 第 2 章 工作原理

### 2.1 GPU 为何适合 AI

深度学习核心是 **大规模矩阵乘法**。GPU 拥有数千个 CUDA 核心，天然适合并行：

```
输入数据 → 权重矩阵 × 激活 → 多层神经网络 → 输出 token/预测
              ↑
         每次运算需从 HBM 读取权重
```

**CUDA**（NVIDIA）是软件生态护城河——模型框架（PyTorch）默认优化 CUDA，形成「硬件 + 软件」双壁垒。

### 2.2 关键术语

| 术语 | 含义 |
|------|------|
| **Fabless** | 无晶圆厂设计商（NVIDIA、AMD、Broadcom） |
| **Foundry** | 晶圆代工（TSMC、Samsung） |
| **CoWoS** | TSMC 2.5D 封装，GPU die + HBM 集成于硅 interposer |
| **HBM** | High Bandwidth Memory，3D 堆叠高带宽内存 |
| **NVLink** | NVIDIA GPU 间高速互连 |
| **InfiniBand** | 数据中心 RDMA 网络，低延迟集群通信 |
| **Reticle limit** | 光刻单次曝光面积上限；Blackwell/Rubin interposer 需 multi-reticle stitching |
| **Memflation** | Gartner 术语：2026 内存价格暴涨 |

### 2.3 从设计到 AI 加速器的完整流程

```
NVIDIA 设计架构
    ↓
TSMC 晶圆制造（4NP/3nm）
    ↓
TSMC CoWoS-L 封装（GPU die + 4~8 颗 HBM）
    ↓
ABF 基板（Ibiden/Unimicron）
    ↓
GPU 模块 → 台湾 ODM 组装 AI 服务器 → 交付 AI 工厂
```

**关键洞察**（TSMC 2026 Q1 财报）：先进封装（CoWoS）产能「非常紧张」，是 **出货量的 binding constraint**——晶圆造出来，包不了，就卖不出。

## 第 3 章 产业链拆解

> **全链条汇总**: 本层完整映射（含 PCB/铜箔/电容等材料子链）见 [10_full_chain_industry_mapping.md §5](10_full_chain_industry_mapping.md#5-第-2-层芯片--全链条映射表)。§8.6 仅为 binding 主线收缩。

### 3.0 全球芯片链总览（含材料挂载点）

```
硅片 → 光刻胶/靶材/CMP → 半导体设备(EUV/刻蚀/沉积)
     → 晶圆制造(TSMC/三星/Intel/中芯)
     → DRAM/HBM → CoWoS/OSAT先进封装
     → ABF载板/类载板 ──┐
     → AI PCB ← CCL ← HVLP铜箔 ←── 材料子链
     → 板级被动件(MLCC/电容) ──────┘
     → GPU/TPU/ASIC/国产算力芯片
     → AI服务器/网络(→ 第3层)
```

### 3.1 上游：半导体设备

| 环节 | 全球龙头 | 瓶颈强度 | 与 AI 的关系 |
|------|----------|----------|-------------|
| EUV 光刻 | ASML | **binding** | 先进 GPU 唯一路径；每台数亿美元 |
| 刻蚀/沉积/CVD/ALD | AMAT、Lam、TEL | **binding** | 产能扩张 18–24 月；扩产周期绑定 AI |
| 检测设备 | KLA、Advantest | structural | 先进制程良率 |
| 清洗/CMP | SCREEN、Entegris | structural | 每道工艺必需 |
| 国产平台型设备 | 北方华创、中微、拓荆 | structural | 管制下国产化 **突破中** |

### 3.2 上游：半导体材料

| 环节 | 全球龙头 | 瓶颈强度 | 与 AI 的关系 |
|------|----------|----------|-------------|
| 硅晶圆 | Shin-Etsu、SUMCO | structural | 12 英寸供给；非当前最紧 |
| 光刻胶 | JSR、TOK、DuPont | structural | EUV 胶高度垄断 |
| 靶材 | JX、Honeywell | structural | 溅射工艺必需 |
| CMP 材料 | Cabot、安集科技 | structural | 平坦化 |
| **ABF 载板 / 类载板** | Ibiden、Unimicron | **structural** | CoWoS 次级瓶颈；扩产慢 |
| **覆铜板 CCL** | 台光电材、生益 | structural | AI 板 Low Dk/Df 材料 |

**材料链说明（为何市场常炒）**：AI Capex 上行 → 服务器/封装板 **量增 + 材料升级** → A 股 PCB/铜箔/电容易成主题。**全球 binding 仍在 CoWoS/HBM**，材料属 **二级传导**（详见 [10 总表 §9](10_full_chain_industry_mapping.md#9-专项深述材料与被动元件)）。

## 第 5 章 瓶颈与约束

### 5.1 CoWoS 封装（当前最紧）

- TSMC CoWoS 产能 CAGR ~80%，仍供不应求
- NVIDIA 预订 2026–2027 大部分产能；AMD、Google 等竞争剩余
- HBM4 需 TSMC 制造 logic base die → 内存更依赖 TSMC

### 5.2 HBM 短缺

- IEA 2026：HBM 短缺或持续至 **2027 年底**
- SK Hynix 主导；Samsung 2026-03 通过 HBM4 认证，双源化起步

### 5.3 地缘与出口管制

**BIS 2026-05-31 官方 guidance**：
- ECCN **3A090.a/b、4A090.a/b** 先进计算芯片
- 向 **Country Group D:5（含中国）或 Macau** 总部/母公司实体出口
- **无论实体位于何处，均需许可证**
- AI Diffusion Rule 暂停执行 **不适用于** 此 2023 年既有要求

### 5.4 台湾集中度风险

- TSMC 制造 ~90% 先进 AI 逻辑芯片
- 台海地缘为 **uncapped tail risk**（供应链分析共识）

## 第 7 章 2025–2030 趋势

1. **Rubin 平台**（2026–2027）— 3nm + HBM4 + 更高 NVLink 带宽
2. **Custom ASIC 占比上升** — Hyperscaler 自研降低 NVIDIA 依赖
3. **推理芯片市场爆发** — 训练 capex 高峰后，推理成为长期算力主需求
4. **Chiplet 架构普及** — 突破 reticle limit，模块化组合
5. **Memflation 2027 后缓解** — Gartner 预计 meaningful 降价 late 2027
6. **出口管制常态化** — 中美芯片脱钩加深

### 8.3 差距分析

| 维度 | 全球领先 | 中国现状 |
|------|----------|----------|

### 8.4 本土替代路径

```
短期：成熟制程 + Chiplet + 云侧集中推理
中期：昇腾/寒武纪在特定场景规模化（政务、运营商）
长期：设备材料突破 + 先进封装 + 国产 HBM
```

**现实判断**：中国路径不是复制 NVIDIA 单卡性能，而是 **「可用够用 + 成本 + 自主可控」** 的组合。

### 8.5 与平安 2019 框架对照

平安证券早期划分 GPU/FPGA/ASIC/芯片 IP——2026 年 ASIC（含 Hyperscaler 自研）和 **Chiplet** 权重显著上升，FPGA 在 AI 训练领域边缘化。

#### 8.6.0 芯片层最高壁垒是什么？（全球 vs 中国 A 股）

**全球**（文档第 5 章）：当前 binding constraint 是 **CoWoS 先进封装 → HBM 供给 → TSMC 先进制程** 三位一体；NVIDIA 占 CoWoS ~60%，HBM 短缺或至 2027 年底（Gartner + IEA）。

**中国 A 股可投资映射**（文档 8.3–8.4）：无法直接持有 TSMC/NVIDIA/SK Hynix，最高壁垒转化为：

```
出口管制下拿不到最先进 GPU/HBM
        ↓
破局顺序：设备/材料（能造）→ 先进封装（能包）→ 国产算力芯片（能用）→ 内存接口（能配）
```

| 壁垒层级 | 全球环节 | 中国 A 股映射方向 |
|----------|----------|-------------------|
| **最高** | CoWoS + HBM + EUV 代工 | **半导体设备** + **先进封装**（「造得出、包得上」） |
| **次高** | CUDA 生态 + 训练集群 | **国产 AI 加速卡**（信创/智算中心准入） |
| **配套** | Memflation / 内存带宽 | **内存接口芯片**（DDR5/HBM 配套，非 HBM 本体） |

#### 8.6.1 筛选准则（五维，与第一层同权重）

| 维度 | 权重 | 芯片层考察什么 | 淘汰谁 |
|------|------|----------------|--------|
| **① 壁垒贴合度** | 30% | 是否直指 **CoWoS 类封装 / 设备 / 国产算力 / 内存接口** 等中国可破局环节 | 仅有「芯片概念」、主业为消费电子 MCU |
| **② 护城河** | 25% | 设备/封装 **市占与认证**；算力芯片 **信创准入与集群落地** | 无规模客户的 IP 初创、同质化封测 |
| **③ 政策确定性** | 20% | 大基金三期、信创、国产算力枢纽 **强制比例** | 政策关联仅停留在「AI 概念」 |
| **④ 业绩可验证** | 15% | 设备 **中标/出货**；封装 **先进封装收入占比**；算力 **DCU/思元出货量** | 长期亏损且无放量信号 |
| **⑤ 可交易性** | 10% | 科创板/主板龙头、流动性 | ST、极小市值 |

**与第一层差异**：能源层 **不选** 燃气链；芯片层 **不选** 纯 FPGA/军工 GPU（AI 智算纯度低）、不选 **无法验证 AI 占比** 的泛 IC 设计。

#### 8.6.3 「最最最」标的（结论）

按 **壁垒—护城河—可验证性** 排序；建议研究顺序即优先级。

---

**🥇 第一优先：北方华创 (002371)** — 打破「造不出来」的最高长周期壁垒

| 项目 | 内容 |
|------|------|
| **对应壁垒** | 全球：EUV/刻蚀/沉积；中国：**设备自主** 是先进制程 + 封装扩产的 **前置条件** |
| **为什么是最优** | 国内半导体设备 **平台型龙头**（刻蚀、CVD、ALD、热处理等）；大基金三期 + 晶圆厂/封测厂扩产 **订单可见**；设备 oligopoly + 长验证周期 = 高护城河 |
| **护城河** | 进入晶圆厂产线后 **切换成本极高**；产品线宽度国内无对手 |
| **验证指标** |  quarterly 新签订单、刻蚀/薄膜设备收入增速；长江存储/中芯等 cap ex |
| **主要风险** | 美国设备管制加码；客户 cap ex 周期性下行 |

---

**🥈 第二优先：长电科技 (600584)** — 中国版「CoWoS 瓶颈」的直接映射

| 项目 | 内容 |
|------|------|
| **对应壁垒** | 全球：**CoWoS 绑定 AI 出货**（文档 5.1）；中国：**先进封装** 是国产 AI 芯片「包不上就卖不出」的同款瓶颈 |
| **为什么是最优** | 全球封测 **前三**；XDFOI / Chiplet / 2.5D 布局最深；直接服务 **昇腾、GPU、CPU** 等算力 die 封装 |
| **护城河** | 先进封装 **资本 + 工艺 Know-how**；客户重测认证周期长 |
| **验证指标** | 先进封装 revenue 占比；算力相关封装订单；产能利用率 |
| **主要风险** | 价格竞争；先进封装扩产不及预期 |

---

**🥉 第三优先：海光信息 (688041)** — 国产算力「能用、能卖、能验证」

| 项目 | 内容 |
|------|------|
| **对应壁垒** | 出口管制下 **先进 GPU 不可得**；信创/智算中心要求 **国产算力占比** |
| **为什么是最优** | 深算 DCU + x86 CPU **双产品线**；政务/互联网智算 **落地案例最多**；相较寒武纪 **④业绩可验证性更强** |
| **护城河** | x86 授权稀缺性；政企 **准入名录**；软件栈持续迭代 |
| **验证指标** | DCU 出货量、智算中心中标、毛利率；与昇腾份额对比 |
| **主要风险** | 单卡性能仍落后 NVIDIA；CUDA 生态迁移成本；授权地缘风险 |

---

**增强配置 A：澜起科技 (688008)** — HBM/DDR5 时代的「内存接口」刚需

| 项目 | 内容 |
|------|------|
| **对应壁垒** | 全球 **HBM 短缺 + Memflation**（Gartner DRAM +125%）；AI 集群 **内存带宽** 为硬约束 |
| **为什么入列** | DDR5 **RCD/DB** 全球龙头；HBM 世代 **接口芯片** 价值量提升；与 SK Hynix 链互补（A 股无法直接买 HBM 原厂） |
| **与前三关系** | 前三是 **造、包、算**；澜起是 **配**（内存配套），受益 memflation 但弹性偏稳健 |
| **主要风险** | 服务器 Capex 波动；竞争加剧 |

---

**增强配置 B：寒武纪 (688256)** — 纯 AI 芯片的高 β 国产化选项

| 项目 | 内容 |
|------|------|
| **对应壁垒** | 国产 **训练/推理 ASIC**；信创算力替代 |
| **为什么入列** | **AI 纯度最高** 的 A 股算力设计标的；思元系列绑定 **互联网 + 智算**；弹性大于海光 |
| **与前三关系** | **进攻型增强**；④验证性弱于海光，适合跟踪 **放量拐点** 而非当期盈利 |
| **主要风险** | 持续亏损；竞争（昇腾生态）；客户集中 |

**未选「增强」但可观察**：中微公司 (688012)（刻蚀设备，与北方华创二选一逻辑）、拓荆科技 (688072)（薄膜/HBM 设备，规模小于平台型龙头）。

## 第 9 章 层间传导

### 向上（芯片 → 基础设施）

- 无芯片，AI 工厂只是空机房
- 800V 机柜、液冷、NVL72 rack 设计 **围绕 NVIDIA GPU 规格**

### 向下（能源 → 芯片）

- B200 单卡 TDP 上升 → 机柜功率密度 ×11（2020–2025）
- 芯片能效提升被 **更大模型、更多卡** 抵消

### 与模型层

- 模型参数 ↑ → 训练需更多 GPU × 更多 HBM
- DeepSeek 等高效模型 → 降低单任务芯片需求，但 **总推理量爆发** 仍推高芯片总需求

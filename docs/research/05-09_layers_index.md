## 层间关系速查

| 传导方向 | 机制 | 典型触发 |
|----------|------|----------|
| 应用 → 模型 | 推理/训练需求 | Agent 普及 |
| 模型 → 基础设施 | 算力租赁/自建 | 大模型 API 调用量 |

| 芯片 → 能源 | 机柜功耗上升 | B200/NVL72 部署 |
| 应用 → 能源（跨层） | Agent 高能耗任务 | IEA：数百~数千倍于简单文本 |

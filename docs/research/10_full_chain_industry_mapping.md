| 等级 | 含义 | 示例 |
|------|------|------|

## 3. 五层总览 ASCII 全链图

```
第1层 能源
  一次能源(气/核/风光) → 发电设备(燃机/SMR) → 电网(特高压/变压器/调度)
       → 园区配电(800V/SiC) → 液冷/散热 → 数据中心用电方
                              ↑
第2层 芯片 ──────────────────┘ (功耗驱动电力需求)
  硅片/材料 → 半导体设备 → 晶圆代工 → HBM/DRAM
       → CoWoS/OSAT封装 → ABF载板 ──┐
       → AI PCB/CCL ← HVLP铜箔 ←────┤ 材料子链（市场常炒）
       → 板级被动件(MLCC/电容) ──────┘
       → GPU/ASIC/国产算力芯片
              ↓
第3层 基础设施
  土地/EPC → 机柜/配电/UPS → AI服务器(ODM/OEM) → 存储/电源模块
       → 交换芯片/DPU → 800G光模块 → 光纤 → IDC/AIDC上架
              ↓
第4层 模型
  数据/语料/标注 → 训练(闭源/开源) → API/MaaS/端侧推理
              ↓
第5层 应用
  Agent/垂直SaaS → 智驾/安防/金融/工业/机器人 → PMF变现 → 向下拉动全栈
```

**材料链挂载点**：PCB/CCL/铜箔/电容主要挂在 **第 2 层（封装基板 + 算力板）** 与 **第 3 层（服务器电源/主板）**；不是五层框架的独立层，但必须在此显式标出。

## 9. 专项深述：材料与被动元件

### 9.1 产业链位置

```
HVLP铜箔 → 覆铜板(CCL) → PCB厂 → { ABF载板→CoWoS | AI服务器板→ODM }
                                    ↓
                              板级贴装 → MLCC/电容/电阻（被动元件）
                                    ↓
                              服务器PSU → 铝电解/固态电容（另一路径）
```

### 9.2 真的很缺吗？

| 子环节 | 判断 | 说明 |
|--------|------|------|
| ABF / 类载板 | **结构性偏紧** | 与 CoWoS 扩产同步；日台龙头主导 |
| AI 服务器/800G 交换板 | **结构性偏紧** | 高层数 HDI + Low Dk 材料；交期拉长 |
| HVLP 铜箔 | **分品种** | 总量不缺；**高端 HVLP/RTF** 偏紧 |
| 普通 PCB / 电池箔 | **不缺** | 产能过剩区与 AI 无关 |
| MLCC / 铝电容 | **周期为主** | 2023–24 曾过剩；AI 增量 < 汽车/消费电子 |

### 9.3 真的有壁垒吗？

| 档次 | 壁垒来源 | 代表 |
|------|----------|------|
| **高** | 客户认证 2–3 年 + 材料 Know-how | ABF 载板、NV 认证 AI 板 |
| **中高** | 材料配方 + 下游绑定 | Ultra Low Loss CCL、HVLP3 铜箔 |
| **中** | 规模 + 良率 | 一般通信 PCB、生益 CCL |
| **低** | 价格竞争 | 低端 PCB、通用 MLCC |

### 9.4 为何市场炒、为何未进 §8.6 主线

| 现象 | 原因 |
|------|------|
| 市场炒 PCB/铜箔/电容 | Capex 上行 → **量增 + 涨价叙事** 易传播；A 股标的多、适合主题轮动 |
| 未进 §8.6 | 全球 **binding** 仍是 CoWoS/HBM/光模块/算电协同；材料属 **二级传导** |
| 研究上必须写 | 否则读者看到行情会质疑框架 **遗漏关键环节** |

# Russshare 人生中枢

## 背景

投资系统很强，但人生不止投资 — 需要统一看财务执行、OKR、健康、决策。

## 做法

- 门户 + 8 模块（finance / invest / goals / learning / career / health / journal / research）
- 5C 复利框架：现金 / 认知 / 能力 / 决策 / 体能
- 静态 GitHub Pages，无后端

## 结果

- 从「LCAI 投资中枢」演进为「Russshare 人生中枢」
- 财务模块外置 config；敏感页 admin gate

## 沉淀技能

信息架构 · 静态站点模块化 · 个人知识系统
